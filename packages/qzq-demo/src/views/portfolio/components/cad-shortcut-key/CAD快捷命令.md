# CAD 快捷命令

### 绘制命令

| 快捷命令 | 功能描述        | 描述                                           |
| :------- | :-------------- | :--------------------------------------------- |
| L        | 直线            | 创建直线段                                     |
| XL       | 构造线          | 创建无限长的线                                 |
| PL       | 多线段          | 创建二维多段线                                 |
| POL      | 多边形          | 创建等边闭合多段线                             |
| REC      | 矩形            | 创建矩形多段线                                 |
| ARC      | 圆弧            | 用三点创建圆弧                                 |
| C        | 圆              | 用圆心和半径创建圆                             |
| DON      | 圆环            | 创建圆对象                                     |
| REVC     | 修订云线        | 通过绘制自由形状的多段线创建修订云线           |
| SPL      | 样式曲线        | 创建通过或接近指定点的平滑曲线                 |
| EL       | 椭圆            | 创建椭圆或椭圆弧                               |
| B        | 块              | 从选定对象创建块定义                           |
| PO       | 点              | 创建多个点对象                                 |
| DDPT     | 点样式          | 修改点样式                                     |
| DIV      | 定数等分        | 沿对象的长度或周长创建等间隔排列的点对象或块   |
| ME       | 定距等分        | 沿对象的长度或周长按指定间隔创建点对象或块     |
| H        | 图案填充/渐变色 | 使用填充图案或填充对封闭区域或选定对象进行填充 |
| HE       | 修改填充/渐变色 | 修改已填充的样式                               |
| MT       | 多行文字        | 创建多行文字对象                               |
| REG      | 面域            | 将包含封闭区域的对象转换为面域对象             |
| WIP      | 区域覆盖/遮罩   | 创建区域覆盖对象                               |
| DR       | 绘图顺序        | 修改绘制顺序                                   |
| RE       | 重生成          | 重新生成当前视图                               |

### 标注命令

| 快捷命令 | 功能描述       | 描述                   |
| :------- | :------------- | :--------------------- |
| D        | 标注样式管理器 | 修改标注样式           |
| DLI      | 线性           | 创建线性标注 水平/垂直 |
| DAL      | 对齐           | 创建对齐线性标注       |
| DAR      | 弧长           | 创建弧长标注           |
| DRA      | 半径           | 创建圆或圆弧的半径标注 |
| DDI      | 直径           | 创建圆或圆弧的直径标注 |
| DAN      | 角度           | 创建角度标注           |

### 修改命令

| 快捷命令 | 功能描述 | 描述                                         |
| :------- | :------- | :------------------------------------------- |
| E        | 删除     | 从图形删除对象                               |
| CO       | 复制     | 将对象复制到指定方向上的指定距离处           |
| MI       | 镜像     | 创建选定对象的镜像副本                       |
| O        | 偏移     | 创建同心圆、平行线和等距曲线                 |
| AR       | 矩形阵列 | 按任意行、列和层级组合分布对象副本           |
| M        | 移动     | 将对象在指定方向上移动指定距离               |
| RO       | 旋转     | 绕基点旋转对象                               |
| SC       | 缩放     | 放大或缩小选定对象，缩放后保持对象的比例不变 |
| S        | 拉伸     | 通过向左绿色窗选或多边形框选的方式拉伸对象   |
| TR       | 修剪     | 修剪对象以适合其他对象的边                   |
| EX       | 延伸     | 延伸对象以适合其他对象的边                   |
| J        | 合并     | 合并相似对象以形成一个完整的对象             |
| F        | 圆角     | 给对象加圆角                                 |

### 常用快捷键

| 快捷键       | 功能描述           |
| :----------- | :----------------- |
| ESC          | 放弃选择,取消命令  |
| enter        | 确定/重复上次      |
| 空格         | 某些时刻等同于回车 |
| Shift + 左键 | 减选               |
| Ctrl + N     | 新建               |
| Ctrl + O     | 打开               |
| Ctrl + S     | 保存               |
| Ctrl + P     | 打印               |
| Ctrl + A     | 全选               |
| Ctrl + C     | 复制               |
| Ctrl + X     | 剪切               |
| Ctrl + V     | 粘贴               |
| Ctrl + Z     | 撤销               |
| Ctrl + Y     | 重做               |
